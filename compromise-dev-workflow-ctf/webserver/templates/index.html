Codetrov


  • Welcome to Codetrov, the place for code coverage offering great insights with visualizations for your project. We're so popular, we have users uploading their test results every 30 seconds!

  • The process is simple. You only need to upload your test results and we'll take care of the rest.

  • To upload your test results, set your $TROVTOKEN environment variable to the token supplied with your account.

  • Next, you will need to set your $TROVUPLOADDIR environment variable. This just tells us what directory contains the test results that you would like to upload and analyze.

  • Lastly, run the following command bash <(curl -s http://codetrov/bash) and we will handle it from there. (Please note depending on the size of the file it may take several minutes to process and analyze.)
